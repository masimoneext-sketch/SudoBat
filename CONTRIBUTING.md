# Contributing to SudoBat

*(Italiano sotto / English first)*

Thanks for your interest! SudoBat is licensed under the **PolyForm Noncommercial License 1.0.0**, and the maintainer reserves all commercial rights (including the ability to offer separate commercial licenses). To keep that legally possible, every contribution must come with two agreements:

1. **DCO (Developer Certificate of Origin)** — sign off every commit with `git commit -s`, certifying the code is yours to contribute (https://developercertificate.org).
2. **Relicensing grant** — by submitting a contribution you agree that it is provided under the project license **and** you grant the maintainer (Marco Simone) a perpetual, irrevocable right to relicense your contribution, including for commercial purposes.

If you are not comfortable with point 2, please open an issue instead of a pull request — ideas and bug reports are always welcome without any paperwork.

Practical notes: run `python3 -m sudobat.selftest` (and `python3 -m sudobat.ui --selftest` on a Batocera box) before opening a PR; keep the code style of the surrounding files; UI strings go through `sudobat/ui/i18n.py` in **both** IT and EN.

## Contribuire a SudoBat (italiano)

SudoBat è rilasciato sotto **PolyForm Noncommercial License 1.0.0** e il maintainer si riserva tutti i diritti commerciali (incluse eventuali licenze commerciali separate). Perché ciò resti legalmente possibile, ogni contributo richiede due accordi:

1. **DCO** — firma ogni commit con `git commit -s`, certificando che il codice è tuo e puoi contribuirlo (https://developercertificate.org).
2. **Concessione di rilicenza** — inviando un contributo accetti che sia fornito sotto la licenza del progetto **e** concedi al maintainer (Marco Simone) il diritto perpetuo e irrevocabile di rilicenziarlo, anche per scopi commerciali.

Se il punto 2 non ti va bene, apri una issue invece di una pull request — idee e segnalazioni sono sempre benvenute senza burocrazia.

Note pratiche: esegui `python3 -m sudobat.selftest` (e `python3 -m sudobat.ui --selftest` su un Batocera) prima della PR; rispetta lo stile del codice circostante; le stringhe UI passano da `sudobat/ui/i18n.py` in **entrambe** le lingue IT e EN.
